This is a demonstration website for the **COS30045 Data Visualisation** unit. It provides information about appliance energy consumption in Australian households and serves as a platform for hosting future data visualisations.

## Features

-   **Interactive Navigation**: Single-page application feel using JavaScript to toggle content sections.
-   **Responsive Design**: layouts that adapt to desktop and mobile screens.
-   **Accessibility**: Use of semantic HTML and ARIA attributes for better screen reader support.
